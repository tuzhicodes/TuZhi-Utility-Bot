TuZhi Utility — Privacy Policy

Last Updated: November 1, 2025

1. Introduction

TuZhi Utility is a multi-purpose Discord bot that provides features such as moderation, automod, fun commands, giveaways, ticketing, and more. We respect your privacy and collect only the data necessary for our services to function properly.

2. Information We Collect

  • Discord User IDs and Server IDs for configuration
  • Server settings, XP, coins, or leaderboard data (if used)
  • No personal information like names, emails, or IPs is stored

3. How We Use Data

Collected data is used solely for bot functionality — e.g., saving settings, executing commands, and improving user experience.

4. Data Storage & Security

Data is securely stored and accessible only to authorized developers. It is deleted automatically when a server removes the bot or upon user request.

5. Data Sharing

We do not share or sell your data to third parties.

6. User Data Deletion

Users can request data removal by joining our Official Discord Server and creating a support ticket, or by contacting us via email at onlyfortuzhi@gmail.com.

7. Policy Updates

This policy may be updated occasionally. Continued use of the bot means acceptance of any future changes.

8. Contact

For privacy-related questions, contact us through our Official Server or email onlyfortuzhi@gmail.com.
